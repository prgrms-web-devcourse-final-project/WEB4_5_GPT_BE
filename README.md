### 📌 1. 프로젝트명
**UniHub — 대학교 통합 관리 서비스**


### 📚 2. 프로젝트 소개
> **UniHub**는 분산된 대학 생활 서비스를 하나의 통합 플랫폼에서 관리할 수 있도록 지원합니다.  
> 수강 신청, 캠퍼스 활동, 전공 관리 등 다양한 기능을 제공하여 학생과 교직원의 대학 생활을 쉽고 편리하게 만듭니다.

### 🚀 3. 주요 기능
- [ ] 수강신청 기능
- [ ] 강의/시간표 관리
- [ ] 전공 변경 기능
- [ ] 회원가입 / 로그인 (이메일 인증)
- [ ] 교수·학생 역할 관리
- [ ] 알림 시스템
- [ ] 마이페이지 (개인정보 수정, 비밀번호 변경)
- [ ] 관리자 기능 (회원 관리, 수강신청 기간 설정 등)
